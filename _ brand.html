Our Design "Source Code"

Round Icons

  • Artboard Size: 1024px x 1024px
  • Outer Circle Diameter: 1024px
  • Inner Circle Diameter: 936px; Border 44px
  • Suggested size for inner symbol: 400px - 800px
  • Dropshadow of symbol: 10px down, 5% blur, opacity 15%
  • Outer Circle/Border Color: Linear Gradient, Top to Down, #F5F4F5 to #D5D5D5
  • Inner Circle Color: Radial Gradient, Top to Down, Color Varies
  • File Format: SVG or Inkscape SVG
  • Credit can be added as comment in SVG

Rectangular Icons

  • Artboard Size: 1024px x 1024px
  • Outer W x H: 895px x 895px
  • Inner W x H: 831px x 831px; Border 32px
  • Icon Rotation: 9° left
  • Outer/Border Color: #FFFFFF
  • File Format: SVG or Inkscape SVG
